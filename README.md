# Install Cypress

[Follow these instructions to install Cypress.](https://on.cypress.io/guides/installing-and-running#section-installing)

# Getting Started

```bash
npm i
```

# Start Cypress

Interactive mode

```bash
npm run cypress:open
```

CI mode

```bash
npm run cypress:run
```

Record results as movies

```bash
npm run cypress:record
```
